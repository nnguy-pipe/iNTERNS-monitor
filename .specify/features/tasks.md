# Feature Specification
## User Scenarios & Testing
### User Story 1
As a developer, I want to accurately simulate CPU/ RAM usage in a lightweight scale. I want the numbers and metrics to accurately mesasure as set by a preset.

This is a priority because I wish to accurately simulate this to be integrated with another product to monitor these changes.

Independent Tests: Create tests to make sure the presets are accurately representing high/ low CPU usage

### User Story 2
As a user, I want to be able to understand the data the simulation displays. I want documentation explaining the significance of the numbers and usage purpose

This is a priotity to allow users to understand how the simulations work
