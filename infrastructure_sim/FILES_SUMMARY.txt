QUICK REFERENCE
===============

To Run Everything:
  1. Start daemon: python3 infrastructure_simulator_daemon.py --port 9999
  2. Start backend: python3 -m src.api.main
  3. Test integration: python3 infrastructure_sim/test_backend_integration.py

To Use Simulator Metrics in Backend Code:
  from src.simulator_bridge import pull_simulator_metric
  event = pull_simulator_metric("my-service", "ci")
  # Send event to /api/events

To Export for Database:
  curl http://localhost:8000/api/simulator/export/json > export.json
  # Import with your SQL loader

STORAGE REQUIREMENTS
====================

Memory (Daemon):
  - 10-20 MB for 3600 ticks (1 hour of metrics)
  - Grows ~0.5 KB per subsystem per tick
  - Automatic pruning after 3600 ticks

Disk (Exports):
  - ~60-80 KB for 2-minute run
  - ~1.8 MB for 1-hour run
  - JSON and XML both included in size estimate

Throughput:
  - Can handle 100+ metric pulls per second
  - Can sustain 1000+ ingest requests per second (backend limited)
  - Export JSON/XML: <500ms for full history
